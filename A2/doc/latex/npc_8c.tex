\doxysubsection{Detailed Description}
NPC drawing and dialog helpers for Maze\+Runner. 

\begin{DoxyAuthor}{Author}
Mohit Saroya Defines ASCII art for a simple NPC (Dave), manages a small NPC window and provides utilities to render the NPC face, a "{}lost"{} indicator, and dialog lines. 
\end{DoxyAuthor}

Draw a simple "{}lost"{} NPC indicator in the maze window. 

Draw a "{}lost"{} NPC at the specified coordinates inside {\ttfamily mazewin}.

The function optionally toggles the {\ttfamily lost\+\_\+state} when {\ttfamily player\+\_\+input} is non-\/zero to animate the indicator between {\ttfamily first\+\_\+y} and {\ttfamily second\+\_\+y} rows. It clears previous characters at the anchor column before drawing the current indicator and refreshes the parent window.

Initialize and return an NPC dialog window sized relative to parent. 

Initialize an NPC window sized relative to the parent.

The dialog height is computed as approximately 15\% of the parent height and is positioned near the bottom of the parent region. The returned window is boxed and refreshed before being returned. Caller is responsible for destroying the window when no longer needed.
