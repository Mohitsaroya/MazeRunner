\doxysubsection{Detailed Description}
Program entry point and top-\/level control flow for Maze\+Runner. 

\begin{DoxyAuthor}{Author}
Mohit Saroya Initializes the ncurses environment, displays the title screen, and runs the main menu loop. The main loop calls the level phases ({\ttfamily level1\+Phase}, {\ttfamily level2\+Phase}) and responds to their return codes ({\ttfamily NEXT}, {\ttfamily MENU}, {\ttfamily QUIT}) to progress, return to menu, or quit. 
\end{DoxyAuthor}
