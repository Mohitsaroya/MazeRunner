\doxysubsection{Detailed Description}
Public interface for player movement utilities. 

Functions and types for creating and updating the player\textquotesingle{}s position in a maze window. Movement helpers update the internal {\ttfamily movement} structure and are responsible for erasing/redrawing the player glyph in the associated {\ttfamily WINDOW} when called.
