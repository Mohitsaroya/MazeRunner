Implementation of high-\/level ncurses UI and menu handling functions. 

\begin{DoxyAuthor}{Author}
Mohit Saroya Provides window creation, title/menu/pause/retry screens, quit confirmation, and goodbye message display using ncurses and the game\+Cards module. 
\end{DoxyAuthor}
