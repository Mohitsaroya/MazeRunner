\doxysubsection{Detailed Description}
Non-\/player character (NPC) utilities and drawing helpers. 

\begin{DoxyAuthor}{Author}
Mohit Saroya Declares layout constants, NPC art data and functions used to initialize an NPC window and draw NPC content such as Dave\textquotesingle{}s face, a "{}lost"{} NPC, and dialog lines. 
\end{DoxyAuthor}

Initialize an NPC window sized relative to the parent. 

Allocates and returns a new ncurses {\ttfamily WINDOW} suitable for drawing NPC art/dialog. The window size is computed from {\ttfamily parent\+\_\+height} and {\ttfamily parent\+\_\+width} (caller still owns and should eventually destroy the window).

Initialize an NPC window sized relative to the parent.

The dialog height is computed as approximately 15\% of the parent height and is positioned near the bottom of the parent region. The returned window is boxed and refreshed before being returned. Caller is responsible for destroying the window when no longer needed.
