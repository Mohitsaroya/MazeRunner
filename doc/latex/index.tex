 A terminal-\/based maze game written in C using ncurses.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{index_autotoc_md1}
{\bfseries{Maze\+Runner}} is an interactive maze game where the player navigates through a maze, encounters obstacles, and attempts to reach the exit. The game features a full menu system, pause functionality, level progression, and retry mechanics-\/all implemented in the terminal using the ncurses library. In the maze you\textquotesingle{}ll find trapped victims of evil person called Dave. Dave has trapped them for years, and if you dont get out, you\textquotesingle{}ll meet the same fate as them; trapped for eternity is dave\textquotesingle{}s mazes, going mad day by day.

The game is divided into 2 levels\+:

{\bfseries{Level 1}}\+: players get to interact with 1 NPC, a poor trapped victim of Dave\textquotesingle{}s sadistic nature. You\textquotesingle{}ll have to get to the door with \$ sign to reach the exit, but wait, Dave has other plans for you.

{\bfseries{Level 2}}\+: After you complete level 1, dave will trap you in a separate maze. You will have to exit the maze through another door with \$ on it before time runs out. If you don\textquotesingle{}t, Dave will destroy the maze, but don\textquotesingle{}t worry as you get unlimited retries to finish the level.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Title and Menu Screens}}\+: ASCII-\/art splash screens and intuitive menu navigation.
\item {\bfseries{Multi-\/\+Level Gameplay}}\+: Multiple levels with increasing difficulty.
\item {\bfseries{Player Movement}}\+: Real-\/time keyboard input for movement (arrow keys).
\item {\bfseries{Pause Menu}}\+: In-\/game pause with options to resume, return to main menu, or quit.
\item {\bfseries{Retry Mechanism}}\+: Players can retry a level on failure.
\item {\bfseries{Quit Confirmation}}\+: Confirmation prompt before exiting the game.
\item {\bfseries{Documented Codebase}}\+: Comprehensive Doxygen documentation for all public interfaces and implementations.
